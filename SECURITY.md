# Security Policy

## Supported Versions


| Version | Supported          |
| ------- | ------------------ |
| 1.3.4   | :white_check_mark: |
| 1.3.3   | :white_check_mark: |
| 1.3.2   | :white_check_mark: |
| 1.3.1   | :x:                |
| 1.2.3   | :x:                |
| < 1.2.3 | :x:                |

## Reporting a Vulnerability

Create a issue with [security] in the title. Describe the issue, and if applicable,
list the line(s) with the issue.
